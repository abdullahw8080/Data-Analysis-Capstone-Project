Data Insights: Global Developer Trends Analysis (2023)
📌 Project Overview
This project is a comprehensive analysis of the 2023 Stack Overflow Developer Survey. The goal is to identify current technology usage, future learning trends, and the socio-economic factors influencing the global developer community. This analysis was completed as part of the IBM Data Analytics Professional Certificate on Coursera.

🚀 Key Insights
Language Dominance: JavaScript and SQL remain essential, but there is a clear shift toward Rust and Go for future projects.
Database Trends: PostgreSQL has emerged as the industry standard for both current use and future desire.
AI Integration: A significant 45% of developers are now utilizing AI tools to augment their coding workflows.
Work Dynamics: Hybrid work models are the most preferred, especially among the 25-34 age demographic.
Economics: Clear correlation between specialized skills (Cloud/AI) and higher compensation tiers.

🛠️ Tools & Technologies Used
Data Processing: Python (Pandas, NumPy)
Data Visualization: Matplotlib, Seaborn
Dashboards: Looker Studio / IBM Cognos
Data Source: Stack Overflow Annual Developer Survey 2023
Additional Data: Job Postings dataset (job-postings.xlsx)
